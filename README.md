# 🚀 SS-2022 多端口轻量管理脚本 (基于 Xray)

专为低配 VPS（落地鸡）打造的超轻量 Shadowsocks-2022 一键管理工具。抛弃臃肿的 Web 面板，极致占用，原生支持多端口分发、防重启流量统计与 Telegram 机器人自动对账！

## ✨ 核心特性

* 🪶 **极致轻量**：底层采用纯净版 **Xray-core**，无面板、无数据库，128M 内存的低配机器也能丝滑运行。
* 🔒 **前沿协议**：自动配置 Shadowsocks 2022 (`2022-blake3-aes-128-gcm`) 协议，一键生成符合规范的 Base64 高强度密钥。
* 👥 **多端口与防冲突**：支持无限添加多端口节点，为每个车友分配完全独立的端口；自带已用端口防呆提示，拒绝冲突。
* 💾 **流量数据持久化**：独创轻量级流量存档机制。VPS 断电、重启**不丢流量数据**，精准记录每个端口的上下行总消耗。
* 🤖 **TG 自动财报 (可自定义)**：内置定时任务，可设置每隔 X 小时自动将各节点流量推送到 Telegram，支持**自定义报表标题**，多机管理不迷路。
* 📅 **定时清零**：支持设定每月指定日期（1-28号）自动重置所有流量数据，方便按月算账。
* ⚡ **全局快捷指令**：首次运行后自动注册系统全局命令，随时随地输入 `ss2022` 即可呼出管理菜单。
* 🔗 **一键分享**：自动生成 `ss://` 标准订阅链接与 Surge 配置规则，极大降低车友导入门槛。
